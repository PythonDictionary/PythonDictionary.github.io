loop

for loop

for i in range(0, 10): # start(optional), end, step(optional)
    print(i, end="")

output:
0123456789

for loop can iterate iterable object(e.g. list, string, sets, range())

range():
- create list of number from start to end...
- it will create list of number in interval <start; end) which means parameter start is included and end is not

example of list

lst_of_color = ["red", "blue", "yellow", "green"]
for color in lst_of_color:
    print(color, end=" ")

output:
red blue yellow green

enumerate

lst_of_color = ["red", "blue", "yellow", "green"]
for idx, color in enumerate(lst_of_color):
    print(idx, color)

output:
0 red
1 blue
2 yellow
1 green
