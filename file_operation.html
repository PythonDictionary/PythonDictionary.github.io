Python File Operations Tutorial

Opening a File

Use the open() function to work with files:

# Open a file in read mode
with open('example.txt', 'r') as file:
    content = file.read()

Reading from a File

Different methods for reading file content:

                # Read entire file
content = file.read()

# Read line by line
for line in file:
    print(line)

# Read all lines into list
lines = file.readlines()

Writing to a File

Writing to files using different modes:

# Write to a new file (overwrites existing)
with open('new_file.txt', 'w') as file:
    file.write("Hello, World!")

# Append to existing file
with open('existing.txt', 'a') as file:
    file.write("\nNew content")
