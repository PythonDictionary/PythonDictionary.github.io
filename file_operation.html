1. Opening file syntax

read

with open("example.txt", "r") as subor:
    m = subor.read():

output:

"040707 45.6 112 Horna Suca\n 040716 34.8 98 Dolna Poruba - Horny Koniec\n 040718 74.3 219 Nimnica\n 040719 54.4 160 Chocholna - dolina\n 040722 131.1 389 Piestany\n 040802 21.7 61 Trencin\n 040804 139.0 430 Piestany"

Prečíta celý dokument a vráti ho ako reťazec so znakov novými riadkami.(\n)

readline

with open("example.txt", "r") as subor:
    m = subor.readlines()

output:

"040707 45.6 112 Horna Suca\n"

precita iba jeden riatok

readlines

with open("example.txt", "r") as subor:
    m = subor.readlines()

output:

['040707 45.6 112 Horna Suca\n', '040716 34.8 98 Dolna Poruba - Horny Koniec\n', '040718 74.3 219 Nimnica\n', '040719 54.4 160 Chocholna - dolina\n', '040722 131.1 389 Piestany\n', '040802 21.7 61 Trencin\n', '040804 139.0 430 Piestany']

precita cely document a vrati list s jednotlivy riatkov
